  • SQL: Writing SQL queries on Spark
  • Data I/O: Reading and writing various data formats
  • Performance: Optimizing Spark applications
  • Advanced: Streaming, ML, and distributed computing

  • 4GB RAM minimum
  • Basic programming knowledge
