🚀 How to Use

View Locally

# Install dependencies
pip install mkdocs mkdocs-readthedocs-theme

# Start local server
cd /home/numrise/databricks-tutorials
mkdocs serve

# Opens at http://localhost:8000

Deploy to GitHub Pages

  1. Push repository to GitHub
  2. Enable GitHub Pages in repository settings:
  3. Settings → Pages
  4. Source: Deploy from a branch
  5. Branch: main (or master)
  6. Folder: public
  7. GitHub Actions workflow automatically:
  8. Builds the site on push
  9. Deploys to public folder

  10. Updates GitHub Pages

  11. Site available at: https://your-username.github.io/databricks-tutorials/

Add New Documentation

  1. Create .md file in appropriate docs/ subfolder
  2. Update navigation in mkdocs.yml
  3. Commit and push
  4. GitHub Actions automatically builds and deploys
