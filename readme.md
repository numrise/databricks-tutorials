# Spark, Databricks & Azure Databricks Tutorials

## Repository Structure

- **[Spark](spark/)**: Apache Spark tutorials and resources.
- **[Databricks](databricks/)**: Databricks platform guides.
- **[Azure Databricks](azure-databricks/)**: Azure-specific Databricks tutorials.

## Getting Started

```bash
git clone https://github.com/your-username/databricks-tutorials.git
cd databricks-tutorials
```

Install dependencies:

```bash
pip install pyspark pandas numpy
```

Set up Docker for local Spark (Optional)

```bash
docker pull bitnami/spark
```
